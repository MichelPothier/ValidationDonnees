# ValidationDonnees
Outils de sélection et de validation des données spatiales. / Tools for selecting and validating spatial data. (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)

BarreEdgeMatch (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)
-Barre de menu utilisée dans dans ArcMap pour valider et corriger l'ajustement entre les éléments à la limite des géométries de découpage des données cartographique (EdgeMatch). 
 Pour plus de détails, voir le fichier de documentation "BarreEdgeMatch.doc" présent dans ce dépôt.
-Menu bar used in in ArcMap to validate and correct the fit between elements at the boundary of the map data geometry (EdgeMatch).
 For more details, see the documentation file "BarreEdgeMatch.doc" present in this repository.

BarreLayerSelection (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)
-Barre de menu utilisée dans dans ArcMap pour effectuer différents traitements d'affichage d'éléments sélectionnés. 
 Pour plus de détails, voir le fichier de documentation "BarreLayerSelection.doc" présent dans ce dépôt.
-Menu bar used in in ArcMap to perform various display processing of selected items.
 For more details, see the documentation file "BarreLayerSelection.doc" present in this repository.

BarreSelection (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)
-Barre de menu utilisée dans dans ArcMap pour effectuer différents traitements de sélection et de validation d'éléments cartographique. 
 Pour plus de détails, voir le fichier de documentation "BarreSelection.doc" présent dans ce dépôt.
-Menu bar used in ArcMap to perform various map selection and validation processes.
 For more details, see the documentation file "BarreSelection.doc" present in this repository.

ValiderContrainte (ArcGis 10.5 / EXE / Vb.Net)
-Programme exécutable indépendant utilisée pour effectuer différents traitements de validation d'éléments cartographique. 
 On retrouve les mêmes fonctionalités de validation dans ce programme que dans la barre de sélection.
 Pour plus de détails, voir le fichier de documentation "BarreSelection.doc" présent dans ce dépôt.
-Independent executable program used to perform different map element validation processes.
 We find the same validation features in this program as in the selection menu bar.
 For more details, see the documentation file "BarreSelection.doc" present in this repository.
